# Tevi Poptracker

Archipelago Tevi tracker pack for [PopTracker](https://github.com/black-sliver/PopTracker/) with Autotracking.

PopTracker v0.23.0 or higher is recommended.

## WIP

+ Location Tracking
+ Map Tracking
+ Shop/Crafting Times
+ SlotData settings updates
